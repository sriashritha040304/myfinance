# MyFinance 💜

A personal finance tracker built with React, Python FastAPI & Supabase.

## Features
- 🛒 Grocery list with expense estimator
- 💸 Bills tracker with month-end summary
- 🎯 Savings goals
- 📊 Daily spending log
- 💳 Debt payoff tracker
- 🧠 Smart expense estimates from history

## Tech Stack
- Frontend: React + Tailwind CSS
- Backend: Python FastAPI
- Database: Supabase (PostgreSQL)
- Hosting: Vercel + Render
